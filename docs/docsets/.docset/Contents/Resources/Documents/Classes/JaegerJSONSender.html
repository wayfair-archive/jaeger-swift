JaegerJSONSender

public final class JaegerJSONSender : JSONSender

A SpanSender designed to send spans at a specified URL using the JSON format. This sender works exactly as a JSONSender, it only modifies the payload sent to the endpoint. The spans will be wrapped in a Batch according to the Jaeger.Thrift definition.

  • init(endPoint:process:session:httpMethod:requestHeaders:)

    Creates a new JaegerJSONSender by specifying the endpoint. This sender works exactly as a JSONSender, it only modifies the payload sent to the endpoint. The spans will be wrapped in a Batch according to the Jaeger.Thrift definition.

    If this basic implementation does not meet your requirements, simply make your own network service (or wrapper) conform to SpanSender.
